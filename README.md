# Frequen-C Mobile

Collaborative music listening app built with Expo / React Native / TypeScript.

**DESN 374-040 @ EWU** — Authority distribution models in collaborative music curation.

## Stack
- Expo SDK 54 + React Native 0.81
- TypeScript strict mode
- Socket.io for real-time sync
- expo-av for audio playback

## Room Modes
- **Campfire** — Equal turns, round-robin queue
- **Spotlight** — Host curates, others suggest
- **Open Floor** — Democratic free-for-all, votes reorder queue
